Title:
Homestyle Interiors – Interior Design Booking Website

Description:
Homestyle Interiors is a web-based application developed using Node.js and MongoDB that allows users to explore interior design styles for different rooms such as hall, kitchen, bedroom, and bathroom. Users can view designs with images, descriptions, and budget details, and book their preferred design by submitting their personal information, which is stored in the database.

Features:
Displays room-wise categories (Hall, Kitchen, Bedroom, Bathroom)
Shows multiple design styles for each category
Detailed view with image, description, and budget
“Book Now” option for each design
Stores user booking details in database
Simple and user-friendly interface

Tech Stack:
Frontend:
HTML
CSS
JavaScript
Backend:
Node.js
Express.js
Database:
MongoDB

Tools Used:
VS Code
MongoDB Compass

Input:
User selects room category
User selects design style
User enters:
Name
Contact Number
Address

Output:
Displays selected design details
Confirms booking submission
Stores booking data in MongoDB database
